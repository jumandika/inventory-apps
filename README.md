## Getting Started

#### 1. Clone and Install

```bash
# Clone the repo
git clone https://github.com/jumandika/inventory-apps.git

# Navigate to clonned folder and Install dependencies
cd inventory-apps && yarn install

# Install Pods
cd ios && pod install
```

#### 2. Open RNS in your iOS simulator

Run this command to start the development server and to start your app on iOS simulator:
```
yarn run:ios
```

Or, if you prefer Android:
```
yarn run:android
```
